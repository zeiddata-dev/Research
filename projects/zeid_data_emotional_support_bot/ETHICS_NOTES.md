# ETHICS NOTES

This bot is designed for emotional literacy and reflective practice.

It should not be used to:

- manipulate someone into contact
- automate deception
- mass generate romance messages
- imitate therapeutic care or crisis support

Healthy target behavior:

- listen
- name emotion carefully
- avoid mind reading
- take accountability
- ask before persuading
- respect silence and boundaries

The story inside the code is about healing. The actual engineering goal is emotional accuracy with restraint.
